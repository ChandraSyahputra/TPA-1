Tentang Saya

Saya merupakan mahasiswa dari Universitas Pembangunan Nasional Veteran Jawa Timur jurusan Informatika dan saat ini sedang menempuh semester 5. Saya juga merupakan salah satu dari peserta studi independen kampus merdeka Skilvul Tech4Impact: Back-end Web Development.

Mengapa membuat website ini

Alasan saya membuat website ini yaitu untuk memenuhi salah satu kriteria penilaian tugas dari program Skilvul yang saya ikuti. Selain itu, saya juga dapat meningkatkan skill saya dengan menerapkan materi yang sudah diajarkan oleh kakak-kakak di Skilvul selama ini yaitu mengenai HTML dan CSS. Website ini juga dapat saya jadikan sebagai website personal (portofolio) saya di masa depan.

Selain itu

Tidak hanya programming, saya juga memiliki beberapa hobi lain seperti workout dan menonton film
